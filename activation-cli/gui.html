🎟️ Generate Token Pembeli

Setiap token unik (Ed25519 + nonce random), ditandatangani oleh creator key, dapat diverifikasi pembeli secara offline. Identitas pembeli tertanam — mencegah distribusi ilegal yang mudah dilacak.

Ini adalah creator key (berbeda dari App Master Key). Generate via menu Creator di mobile app → Salin private key.
Harus sama persis dengan bundleId saat membuat bundle.
Tertanam di token, tidak bisa diubah pasca-penerbitan.
Cara kerja sistem token pembeli:
  1. Creator membuat bundle dengan password tertentu di app mobile.
  2. Creator generate token unik per pembeli di sini (tool ini).
  3. Pembeli menerima: bundle JSON + token JSON + password.
  4. App pembeli verifikasi token (offline) → decrypt bundle → belajar.
  5. Token tidak bisa dipalsukan tanpa private key creator.
  6. Identitas pembeli di token memudahkan pelacakan bocor.
